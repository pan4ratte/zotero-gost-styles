Данный репозиторий представляет собой авторский проект, главной идеей которого является доработка и разработка стилей цитирования для Zotero по ГОСТ. В дальнейшем планируется самостоятельная разработка новых стилей с нуля.

Сейчас в библиотеке только один стиль, который постепенно дорабатывается. Он основан на уже существующем стиле, разработанном Романом Распоповым по ГОСТ 2008 года (см. ниже), но значительно доработан:

1. Стиль цитирования изменён на постраничные сноски.

2. Корректно обрабатывается библиография (т.е. технические термины) источников на русском, английском, немецком и французском языках. Для этого в Zotero в источнике в поле "Язык" достаточно указать значение `ru`, `en`, `de` или `fr`.

3. При цитировании в документе, автоматически обрабатывается повторяющиеся цитирования, причём с учётом языка. Т.е. корректно используются "Ibid" и "Там же" и т.д., в том числе с указанием страниц.

4. При первом цитировании источника выводится его полная библиография, а всех последующих (не Ibid.) — сокращённая (автор + название). Не применяется к веб-страницам.


## Краткосрочные цели

- [x] Улучшить повторяющиеся цитирования таким образом, чтобы полная библиография источника в сноске выводилась только первый раз, а далее выводилась сокращённая версия.

- [x] Избавиться от дублирования имени автора после названия работы при цитировании.
