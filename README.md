## Сотрудничество

Я открыт к сотрудничеству, и в перспективе надеюсь, что данный репозирорий станет большим сборником стилей ГОСТ для Zotero. К сожалению, этот важнейший инструмент пока ещё не раскрылся полноценно в контексте России из-за слабой поддержки сообществом и относительно быстрого устаревания стилей.
