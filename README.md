# Стили Zotero по ГОСТ

Данный проект является авторской разработкой Ингрэма М. А. Его изначальная цель: доработка существующих стилей Zotero, ориентированных на требования ГОСТ. В дальнейшем планируется самостоятельная разработка новых стилей с нуля.

Сейчас в репозитории только один стиль, который постепенно дорабатывается. Он основан на уже существующем стиле, разработанном Романом Распоповым по ГОСТ 2008 года (см. ниже), но значительно доработан:

1. Стиль цитирования изменён на постраничные сноски.

2. Корректно обрабатывается библиография (т.е. технические термины) источников как как на русском, так и на английском языках. Для этого в поле "Язык" источника в Zotero достаточно указать значение `en` или `ru`.

3. При цитировании в документе, автоматически обрабатывается повторяющиеся цитирования, причём с учётом языка. Т.е. корректно используются "Ibid" и "Там же", в том числе с указанием страниц.


## Краткосрочные цели

- [ ] Улучшить повторяющиеся цитирования таким образом, чтобы полная библиография источника в сноске выводилась только первый раз, а далее выводилась сокращённая версия.

- [ ] Избавиться от дублирования имени автора после названия работы при цитировании.


## Долгосрочные цели

- [ ] Разработать стиль по ГОСТ 2018.

- [ ] Переписать с нуля стиль по ГОСТ 2008.
